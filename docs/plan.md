# 基于 nnU-Net 与 U-Mamba 混合架构的前列腺 mpMRI 癌灶/可疑病灶分割研究预方案

## 一、研究题目

**基于 nnU-Net 与 U-Mamba 混合架构的前列腺多参数 MRI 癌灶/可疑病灶自动分割方法研究**

## 二、研究背景与意义

### 2.1 临床背景
前列腺癌（Prostate Cancer, PCa）是全球男性泌尿生殖系统中最常见的恶性肿瘤之一。随着医学影像技术的发展，基于多参数磁共振成像（mpMRI）的病灶自动分割技术在前列腺癌的辅助诊断、精确病灶定位、穿刺引导以及治疗效果评估中扮演着日益关键的角色。然而，相较于前列腺整体器官或腺体区域的分割，癌灶及可疑病灶（Region of Interest, ROI）的自动分割面临着更高的技术挑战。

### 2.2 技术难点
本任务的核心难点主要体现在以下三个维度：
1.  **目标特性复杂：** 癌灶目标体积通常较小，且边界模糊不清，极易受到周围背景组织（如良性增生、炎症等）的干扰，导致分割精度下降。
2.  **多模态互补性建模难：** 不同成像模态（如 T2WI, DWI, ADC）之间存在显著的成像差异和信息互补性，网络需要有效建模多模态间的互补信息以提升特征表达能力。
3.  **空间各向异性显著：** 医学 MRI 数据往往存在明显的空间各向异性，尤其在 z 轴方向层厚较大。直接采用强调全局建模的激进结构（如纯 Transformer）可能会削弱对局部微小病灶的刻画能力，导致细节丢失。

上述问题决定了本任务更适合采用**“局部细节提取 + 适度全局建模”**的混合式网络设计思路。

### 2.3 方法论基础
*   **nnU-Net：** 作为当前医学图像分割领域公认的强基线框架（Strong Baseline），nnU-Net 能够根据数据集自动配置预处理、网络结构和训练流程。官方将其定位为方法开发框架，提供了极高的稳健性。
*   **U-Mamba：** 该架构官方实现直接建立在 nnU-Net 框架之上，提供了 2D/3D fullres 的训练入口以及 `UMambaBot`、`UMambaEnc` 等训练器。其结合了 CNN 的局部感知能力与 Mamba（状态空间模型）的长程依赖建模能力，非常适合作为本研究的结构改进基础。
*   **公平比较原则：** nnU-Net 官方提供了 `residual encoder` 预设，并建议在提出新方法时进行公平比较；U-Mamba 官方也明确提示其可直接用于自定义数据集训练，并提醒 AMP（自动混合精度）在 Mamba 模块中可能出现 NaN，需要预留 `NoAMP` 备选方案。

因此，本研究拟以 **nnU-Net 为稳健底盘**，在其上引入 **U-Mamba 风格的混合式长程依赖建模模块**，并结合**小病灶友好的训练策略**，构建一个更适合前列腺 mpMRI 癌灶/可疑病灶分割的数据驱动模型。

## 三、数据基础与任务特点

### 3.1 数据集概况
本研究采用已完成清洗与筛选的前列腺多模态 MRI 数据集，具体统计如下：
*   **总样本量：** 共 388 例。
*   **病例分布：** 良性前列腺增生（BPH）197 例，前列腺癌（PCA）191 例。
*   **模态完整性：** ADC、DWI 与 T2_not_fs 三种核心模态均为 388/388 完整无缺失，确保了多模态输入的可行性。

### 3.2 空间与强度属性分析
*   **空间形状（Shape）：** 数据的主流 shape 集中于 `(220, 120, 23)` 与 `(220, 120, 20)`。
*   **体素间距（Voxel Spacing）：** 分布存在多个簇，例如 `(0.9091, 0.9091, 3.0)`、`(0.8636, 0.8636, 4.4)`、`(0.8636, 0.8636, 4.8)` 和 `(1.0, 1.0, 3.0)`。这说明数据具有明显的 **z 轴各向异性**，需要在预处理或网络结构中予以考虑。
*   **强度范围：** 图像强度范围中出现 `-32768` 至 `32767` 一类极值，提示存在哨兵值（Sentinel Values）或异常背景，需要在训练前进行稳健的归一化处理。

### 3.3 标签质量
*   **ROI 标签：** 虽然大多数病例为标准二值标签，但仍有少量病例存在多标签/非二值值域。因此，在正式训练前必须进一步复核并修正，以确保监督信号的准确性。

> **数据Profile分析记录：**
> *   `filtered_data_profile_summary` (初步清洗统计)
> *   `filtered_data_profile_summary` (模态完整性校验)
> *   `filtered_data_profile_summary` (空间属性分布确认)

基于以上数据特征，可以判断：**本课题的核心难点并不在于简单的网络替换，而在于如何在保证稳健性的前提下，同时兼顾局部细节保持、长程依赖建模和小病灶敏感性提升。**

## 四、研究目标

本研究拟围绕前列腺 mpMRI 癌灶/可疑病灶自动分割任务，完成以下具体目标：

1.  **构建强基线系统：** 构建一个基于 nnU-Net 的强基线分割系统，并完成标准化的数据预处理、训练与评估流程，确立性能基准。
2.  **引入混合架构：** 在 nnU-Net 框架内引入 U-Mamba 风格的混合网络结构，探索 CNN 局部特征提取与 Mamba 长程建模结合的可行性与优势。
3.  **优化训练策略：** 针对癌灶体积小、边界模糊、类别不平衡等问题，引入小病灶友好的训练策略，显著提升模型对微小病灶的检出与分割能力。
4.  **验证与评估：** 通过定量对比、消融实验和可视化分析，验证所提方法相较于强基线和已有混合模型的有效性与合理性，确保结论可信。

## 五、研究思路与总体方案

### 5.1 总体技术路线
本研究不采用“完全重写分割框架”的方式，而是采用 **"nnU-Net 壳子 + 混合式网络引擎”** 的路线：
*   **保留部分：** 保留 nnU-Net 的数据分析、预处理、patch cropping、数据增强、推理和后处理流程。
*   **改造部分：** 仅对网络主干（Backbone）进行结构改造。

这样既能继承 nnU-Net 的成熟工程优势（如自动适配数据特性），又能使改进点集中体现在网络结构和训练策略上，便于开展公平对比与消融实验。U-Mamba 官方代码本身就延续了 nnU-Net 数据准备与训练接口，因此这一技术路线具备较强可行性。

### 5.2 核心创新点收缩
与原始预方案相比，本研究将创新点收缩为两部分，以确保研究聚焦：
1.  **核心结构创新：** 在 nnU-Net/ResEnc 主干上引入 U-Mamba 风格的 Mamba 模块。优先放置于 **bottleneck** 或 **encoder 深层**，以实现“浅层卷积保局部、高层 Mamba 建全局”的混合式建模。
2.  **小病灶友好策略：** 例如 `lesion-aware patch sampling`（病灶感知 Patch 采样）、`hard negative sampling`（困难负样本采样），或 `Dice + Focal/Tversky` 的小目标重加权损失。

这种设计既符合前列腺小病灶分割的任务特性，也能避免一次性叠加过多改动带来的训练不稳定与实验解释困难。

## 六、研究内容

### （一）强基线构建
首先，基于 **nnU-Net v2** 构建本任务的强基线。具体包括：
1.  **数据清洗：** 对现有 `FilteredData` 进一步检查标签质量，重点清理剩余的非二值 ROI 病例，确保 Ground Truth 纯净。
2.  **自动计划：** 采用 nnU-Net 的数据指纹分析与自动计划生成流程，完成 `plan_and_preprocess`，生成适合本数据集的配置。
3.  **模型训练：**
    *   训练标准 **nnU-Net 3d_fullres** 模型。
    *   进一步训练 **nnU-Net ResEnc** 作为更强的 CNN 基线。
4.  **基准评估：** 在 5-fold 交叉验证基础上统计平均性能与标准差，作为后续所有结构改进的比较基准。
    *   *注：之所以加入 ResEnc 基线，是因为 nnU-Net 官方已提供 residual encoder 预设，并将其视为更强的标准化比较对象，尤其适合新方法的公平评估。*

### （二）U-Mamba 风格混合架构引入
在基线基础上，本研究拟优先引入 **U-Mamba_Bot** 风格的轻改造结构。具体思路为：
1.  **框架保留：** 保留 nnU-Net 的整体 encoder-decoder 框架，确保流程兼容性。
2.  **浅层卷积：** 浅层继续使用卷积模块，以稳定提取局部纹理、边缘与细粒度空间信息。
3.  **深层 Mamba：** 在 bottleneck 或 encoder 深层引入 Mamba 模块，用于建模跨切片和跨区域的长程依赖。
4.  **渐进式改造：** 初期仅替换一个关键层级，避免一次性重构全网络，确保结果可解释、风险可控。若该结构能够带来稳定提升，再考虑扩展到 encoder 深层多层注入的版本。

该方案的核心并非“追求最复杂的 Mamba 结构”，而是通过最小改动验证：**在前列腺癌灶小病灶分割任务中，适度的全局建模是否能够在不破坏局部细节的前提下带来性能提升。**

### （三）小病灶友好的训练策略设计
考虑到目标病灶往往体积较小，标准 patch 采样和常规 `Dice + CE` 损失可能不足以稳定学习前景区域。因此，本研究拟加入一个小病灶友好的训练增强策略，优先从以下两类方案中选择一种作为主策略：
1.  **采样策略优化：** `Lesion-aware patch sampling / hard negative sampling`。提高含病灶 patch 的采样比例，同时从高疑似背景区域中引入困难负样本，减少漏检与误检。
2.  **损失函数优化：** `Small-lesion-aware loss`。在 Dice 基础上引入 Focal、Tversky 或边界辅助项，增强对小目标和模糊边界区域的梯度响应。

本研究将坚持 **“一次只加一个训练创新”** 的原则，以保证消融实验清晰、论文叙事简洁。

## 七、技术路线

本研究的技术路线遵循以下流程：

1.  **数据复核与标准化**
    *   清理非二值 ROI、确认多模态与标签匹配、处理强度异常值。
    *   输出：标准化 nnU-Net 数据集。
    *   $\downarrow$
2.  **强基线训练**
    *   训练 `nnU-Net 3d_fullres`。
    *   训练 `nnU-Net ResEnc`。
    *   输出：基线性能指标。
    *   $\downarrow$
3.  **混合架构训练**
    *   `U-Mamba_Bot` 风格网络接入 nnU-Net。
    *   解决环境与数值稳定性问题。
    *   输出：结构改进版性能。
    *   $\downarrow$
4.  **主方法构建**
    *   在 U-Mamba 风格架构基础上加入一个小病灶友好训练策略。
    *   输出：最终模型（Ours）。
    *   $\downarrow$
5.  **结果分析**
    *   5-fold 定量对比。
    *   消融实验。
    *   定性可视化与失败案例分析。

## 八、实验设计

### 1. 对比实验
为保证实验结果具有说服力，定量实验将采用**同一份数据划分、同一套预处理与评估流程**，主要比较以下模型：
*   **nnU-Net 3d_fullres：** 基础强基线。
*   **nnU-Net ResEnc：** 更强的 CNN 基线。
*   **U-Mamba_Bot：** 验证 CNN + Mamba 混合结构在本任务中的有效性。
*   **Ours：** 在 U-Mamba 风格结构基础上加入小病灶友好策略后的方法。

*注：如算力允许，可补充一个 Transformer 代表模型（如 Swin-UNETR）作为补充对照，但不作为首要任务。*

### 2. 消融实验
为了验证各模块的独立贡献，消融实验建议收缩为以下三组，以避免实验表格过大、结论不清晰：
1.  **Baseline：** 基础 U-Mamba 结构。
2.  **+ 结构改进：** 例如各向异性友好的 Mamba 注入方式。
3.  **+ 小病灶训练策略：** 完整模型（Ours）。

### 3. 评价指标
本研究将采用**体素级与病灶级相结合**的评价方式，以全面评估模型性能：
*   **Dice Score：** 衡量体素重叠率。
*   **95% Hausdorff Distance (HD95)：** 衡量边界质量。
*   **Sensitivity / Recall：** 衡量病灶检出能力。
*   **Lesion-wise sensitivity：** 病灶级灵敏度。
*   **False Positives per case (FP/case)：** 每病例假阳性数。

其中，Dice 与 HD95 用于衡量体素重叠与边界质量；病灶级 sensitivity 与 FP/case 更适合评价小病灶分割的临床实用价值。

### 4. 可视化分析
定性实验将选取具有代表性的病例，包括：
*   极小病灶病例；
*   边界模糊病例；
*   多模态信息存在明显差异的病例。

通过展示分割结果切片图、预测与真值重叠图、以及中高层特征响应热力图，分析模型对病灶区域的关注能力及其优势来源。

## 九、预期创新点

本研究拟形成以下三个层面的创新或贡献：

1.  **任务层面：** 面向前列腺 mpMRI 癌灶/可疑病灶分割这一**小目标、高各向异性场景**，构建适合该任务的稳健分割方案，填补特定场景下的方法空白。
2.  **结构层面：** 在 nnU-Net 强基线框架内引入 **U-Mamba 风格的混合式长程建模模块**，实现局部卷积与全局状态空间建模的结合，探索 SSM 在医学小目标分割中的潜力。
3.  **训练层面：** 设计一个针对**小病灶分割的采样或损失优化策略**，以提高病灶检出率并减少漏分割，提升模型的临床可用性。

需要强调的是，本研究的目标不是提出一个过于复杂的全新框架，而是在强基线之上做**小而有效、可复现、可解释**的改进。

## 十、可行性分析

1.  **数据基础：** 本研究已有 388 例经过清洗的数据，且三模态完整，足以支撑 5-fold 交叉验证，数据规模符合硕士研究要求。
2.  **方法基础：**
    *   nnU-Net 已可作为开箱即用的强基线。
    *   U-Mamba 官方仓库也已给出基于 nnU-Net 的训练方式，包括 `nnUNetTrainerUMambaBot` 和 `nnUNetTrainerUMambaEnc`，因此方法接入难度相对可控。
3.  **环境与实施：**
    *   考虑到 U-Mamba 官方指出 AMP 可能导致 Mamba 模块产生 NaN，且环境依赖 `CUDA` 与 `mamba-ssm` / `causal-conv1d`。
    *   本研究将优先在 **Linux/WSL2** 环境下完成模型编译与训练。
    *   预留 **NoAMP** 训练方案作为备选，确保训练稳定性。

总体来看，本课题在数据规模、方法实现与论文可写性方面均具有较高可行性，适合作为硕士阶段二区论文方向推进。

## 十二、风险与备选方案

为确保研究顺利推进，针对潜在风险制定以下备选方案：

1.  **训练稳定性风险：**
    *   *风险：* 若 U-Mamba 模块在当前环境下训练不稳定（如出现 NaN）。
    *   *对策：* 优先切换至官方 **NoAMP 训练器**，或降低学习率、使用梯度裁剪。
2.  **性能提升风险：**
    *   *风险：* 若 Mamba 结构提升不明显，无法超越 CNN 基线。
    *   *对策：* 优先保留 **nnU-Net ResEnc** 作为主线，并将“小病灶训练策略”作为主要创新点，强调策略的有效性。
3.  **算力资源风险：**
    *   *风险：* 若训练资源不足以支持过多对比模型。
    *   *对策：* 优先保证 **nnU-Net standard / ResEnc / U-Mamba / Ours** 四组实验完整，减少次要对照模型数量。
4.  **多模态融合风险：**
    *   *风险：* 若多模态联合效果不稳定。
    *   *对策：* 先维持 **early fusion** 输入方式，不急于增加复杂的交叉门控模块，确保基础融合有效。
